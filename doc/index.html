Module globalfunc

cropImage (imgpath, cropX, cropY, cropWidth, cropHeight)
Crop images. I'm using this instead of Love2D quads.

Parameters:

  • imgpath Path of the image
  • cropX X position where we'll start to crop the image (in pixels)
  • cropY Y position where we'll start to crop the image (in pixels)
  • cropWidth Width of the cropped image (in pixels)
  • cropHeight Height of the cropped image (in pixels)

Returns:

    Cropped image.
addBlock (id, x, y, w, h, drawable, collidable, execOnInteract)
Add item to the current level.

Parameters:

  • id The ID that we will use to search the item
  • x X coordinates
  • y Y coordinates
  • w Width
  • h Height
  • drawable Sprite that will be used for the item
  • collidable Collide with player (default: true)
  • execOnInteract Function to execute when player interacts with the item

Returns:

    Item that has been created.

moveBlock (id, goalx, goaly)
Move item by ID.

Parameters:

  • id The ID that we will use to search the item
  • goalx Target position in X coordinates
  • goaly Target position in Y coordinates

Returns:

  1. New X value.
  2. New Y value.
  3. Items that are colliding.
  4. Number of items that collided.

SetPlayerDefaultSettings ()
Set player settings to the default ones. The player will be:
  • Able to move
  • Able to interact
  • Able to open the inventory
  • Able to open the menu
  • Visible
LoadLevel (lvl)
Load levels from lua files. Inside the level file put a load() and an update(dt) function

Parameters:
